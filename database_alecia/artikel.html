Belajar HTML & CSS dari Nol: Pengalaman Pertama Membuat Website

Alecia Poppy Mei 2025 Web Development

Pertama kali saya mendengar kata "HTML", yang ada di bayangan saya cuma sekumpulan kode aneh yang nggak ada artinya. Waktu itu saya masih kelas X, baru masuk jurusan RPL dan belum tahu sama sekali soal pemrograman web. Tapi setelah beberapa bulan belajar — dan pastinya banyak trial-error — akhirnya saya mulai paham bahwa HTML itu ternyata tidak sesusah yang dibayangkan. Bahkan bisa dibilang, HTML adalah bahasa pemrograman yang paling ramah untuk pemula.

HTML atau HyperText Markup Language adalah bahasa dasar yang digunakan untuk membangun struktur halaman web. Sederhananya, kalau website itu ibarat sebuah rumah, maka HTML adalah rangka atau fondasi bangunannya. Setiap elemen yang kita lihat di sebuah halaman web — judul, paragraf, gambar, tombol, tabel — semuanya dibuat menggunakan tag-tag HTML. Dan yang lebih menarik lagi, kita bisa langsung melihat hasilnya di browser tanpa perlu install apapun yang rumit.

Apa Itu Tag HTML dan Cara Kerjanya?

Tag HTML ditulis di antara tanda kurung siku, misalnya <p> untuk paragraf atau <h1> untuk judul utama. Setiap tag pembuka biasanya punya pasangan tag penutup yang ditandai dengan garis miring, seperti </p>. Konten yang ingin ditampilkan ditulis di antara kedua tag tersebut. Berikut contoh sederhana yang pertama kali saya coba waktu belajar:

<!DOCTYPE html>
<html lang="id">
  <head>
    <title>Website Pertamaku</title>
  </head>
  <body>
    <h1>Halo, ini website pertama saya!</h1>
    <p>Saya baru belajar HTML.</p>
  </body>
</html>

Waktu kode ini berhasil tampil di browser dan saya lihat tulisan "Halo, ini website pertama saya!" muncul di layar — rasanya luar biasa. Padahal kodenya cuma belasan baris, tapi perasaan senangnya tidak bisa dijelasin dengan kata-kata. Dari situ saya semakin penasaran dan terus belajar lebih jauh.

Kenalan dengan CSS: Mengubah Tampilan Jadi Lebih Cantik

Kalau HTML adalah rangka rumah, maka CSS (Cascading Style Sheets) adalah cat dinding, furnitur, dan dekorasinya. CSS bertugas mengatur tampilan visual dari elemen HTML — mulai dari warna, ukuran font, jarak antar elemen, sampai layout halaman secara keseluruhan. Tanpa CSS, website akan terlihat polos dan membosankan seperti dokumen teks biasa.

Yang paling saya suka dari CSS adalah fitur Flexbox dan Grid. Dua teknik ini memungkinkan kita mengatur posisi elemen dengan sangat fleksibel. Misalnya, navigasi horizontal di bagian atas website ini dibuat menggunakan Flexbox — cukup tambahkan display: flex pada kontainernya, dan elemen-elemen di dalamnya otomatis tersusun secara horizontal. Sederhana tapi powerful banget.

Tips untuk Teman-Teman yang Baru Belajar Web

1. Mulai dari yang Kecil

Jangan langsung coba buat website yang kompleks. Mulai dari halaman sederhana dulu — satu file HTML dengan beberapa tag dasar. Setelah itu tambahkan satu per satu fitur yang kamu pelajari. Progres kecil yang konsisten lebih baik daripada buru-buru tapi hasilnya nggak dipahami.

2. Jangan Takut Error

Error adalah bagian dari proses belajar. Waktu belajar, saya pernah frustrasi karena layout berantakan gara-gara lupa tutup satu tag. Tapi justru dari situ saya jadi lebih teliti dan hafal struktur HTML dengan lebih baik. Setiap error yang berhasil dipecahkan itu sebenarnya adalah ilmu baru yang kita dapatkan.

3. Latihan dengan Proyek Nyata

Teori saja tidak cukup. Saya baru benar-benar paham HTML dan CSS setelah mulai membuat proyek sendiri, termasuk website ini. Coba buat website profil diri, halaman biodata, atau portofolio sederhana. Dengan punya tujuan yang konkret, proses belajarnya jadi jauh lebih menyenangkan dan terasa bermakna.

Belajar web development memang butuh waktu dan kesabaran, tapi hasilnya sangat memuaskan. Saya sendiri masih banyak yang harus dipelajari — JavaScript, PHP, framework, dan banyak lagi. Tapi yang penting adalah tidak berhenti dan terus berlatih. Semangat untuk teman-teman yang juga sedang belajar!
